# Imgur Album Downloader

Command-line downloading of Imgur albums.

## Usage

    bash imgur.sh [album]

or (with execute permission):

    ./imgur.sh [album]

## Example

    ./imgur.sh adkET

or 
    ./imgur.sh http://imgur.com/a/adkET
