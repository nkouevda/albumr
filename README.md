# albumr

Command-line downloading of Imgur albums.

## Usage

```bash
python3 albumr.py [-h] [-n] [-t] [-v] album [album ...]
```

or (with execute permission):

```bash
./albumr.py [-h] [-n] [-t] [-v] album [album ...]
```

## Examples

```bash
./albumr.py adkET
```

or

```bash
./albumr.py http://imgur.com/a/adkET
```
