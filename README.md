# Imgur Album Downloader

A basic Python utility to download Imgur albums.

### To run:

    python3 download.py [album]

or (with execute permission):

    ./download.py [album]

### Example:

    $ ./download.py adkET
    downloading: adkET/BDvh1.png
    ...
    downloading: adkET/sy282.jpg
    saved 27 of 27 images
