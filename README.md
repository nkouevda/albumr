# albumr

Command-line downloading of Imgur albums.

## Usage

```bash
bash albumr.sh [-h] [-v] album ...
```

or (with execute permission):

```bash
./albumr.sh [-h] [-v] album ...
```

## Options

    -h

Show the help message.

    -v

Enable verbose output.

## Examples

```bash
./albumr.sh adkET
```

or

```bash
./albumr.sh http://imgur.com/a/adkET
```
