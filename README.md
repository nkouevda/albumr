# albumr

Command-line downloading of Imgur albums.

## Usage

```bash
python3 albumr.py [-h] [-n] [-t] [-v] album [album ...]
```

## Examples

```bash
python3 albumr.py adkET
```

```bash
python3 albumr.py http://imgur.com/a/adkET
```
